Place menu item images in this folder using the auto-generated item slug names.
Example:
Chicken Steam Momo -> chicken-steam-momo.webp
Veg Chowmein -> veg-chowmein.webp

Recommended image size: 800x600px or any 4:3 crop, WebP format.
If an image is missing, the menu card will show the existing themed gradient fallback.
